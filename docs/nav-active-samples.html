Navigation Active State Samples

Five active-tab treatments for the current soft glass portfolio navigation. These are mockups only.

1. Black Active Pill

Highest contrast. Clear black background with white text, still compact enough for the current nav.

2. Raised Glass Pill

Most aligned with the existing glass UI. Active state is brighter, elevated, and easier to see.

3. Dark Glass Bar

Black and white treatment. Strong visibility, but changes the nav personality more dramatically.

4. Glass With Ink Underline

Quiet and portfolio-like. Keeps the airy nav while giving the active item a visible anchor.

5. Soft Accent Pill

Uses a restrained blue tint from the site palette. Visible without feeling too heavy.
